Architects of Play: Discovering Soccer Passing Archetypes

Brief Description
This project applies the Knowledge Discovery in Databases (KDD) process to English Premier League (EPL) data to uncover hidden player roles. Rather than using traditional positions like "Midfielder" or "Defender," we use unsupervised clustering (K-Means) and association rule mining (Apriori) to discover "Passing Archetypes." The project specifically explores how player behavior shifts based on match-day context, such as opponent pressing intensity and game state, during the 2024-25 and 2025-26 seasons.

Dataset Source
The data is sourced from **FBref** and extracted using the soccerdataPython library.
